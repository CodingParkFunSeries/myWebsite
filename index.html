What is a open source project ?

As this question wasn't tagged "Programming," I'll assume you have no knowledge of computer programming, either. Computer programs are created by writing source code in a very rigorous format akin to natural languages (except, as I said, more rigorous). The source code can be interpreted by a special computer program and turned into a format the computer's processor can understand naturally (called a binary or executable). When programs are written by software companies for money, they typically keep the source code secret and only release the executable files so that people can't change the source code to circumvent security or anti-counterfeit measures. Open source projects, on the other hand, release the source code to the public so that anyone can study it, see how it works, and change and redistribute it if they want to. Programs which are released under these terms are collectively known as "Free/Open Source Software" or F/OSS. You know that "End-user license agreement" you always have to agree to in order to install software? The open source community has over time developed a number of standardized licenses which enumerate the different combinations of rights users have to the software. Some licenses are more permissive than others, and various organizations disagree on what, exactly, constitutes "open source." For example, some licenses still allow selling the software for money, some licenses place restrictions on how different programs can be combined depending on the license of the other program, etc. Some organizations believe that software isn't open source unless it falls under some sort of "copyleft," i.e. derivative software must be released under an equivalent license. However, most of them at least agree that simply releasing the source code isn't enough to be open source. The user must be allowed to make changes and redistribute the software.
